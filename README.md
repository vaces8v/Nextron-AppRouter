## Как использовать?

### Установите зависимости

```
$ cd my-app

# если используете yarn
$ yarn (or `npm install`)

# если используете  pnpm
$ pnpm install --shamefully-hoist
```

### Чтобы запустить

```
# режим разработки
$ yarn dev (или `npm run dev` или `pnpm run dev`)

# режим сборки
$ yarn build (или `npm run build` или `pnpm run build`)
```
